Setup Instructions

Receive timer updates in your Discord server.

  1. Open Discord and go to the channel where you want notifications
  2. Click the gear icon next to the channel name â†’ Integrations
  3. Click Create Webhook or View Webhooks

  6. Add the webhook URL to your server's .env file as DISCORD_WEBHOOK_URL
  7. Restart the server
